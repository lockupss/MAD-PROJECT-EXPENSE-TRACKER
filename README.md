Expense Tracker App

Simple, Smart, Offline Expense Management. project Description
The Expense Tracker App is an Android-based mobile application designed to help users record, monitor, and manage daily financial activities easily. It operates entirely offline using SQLite, ensuring both quick access and complete data privacy.

This lightweight, user-friendly app enables users to add, edit, delete, and view expenses, calculate total spending for weekly or monthly periods, and organize transactions by category.
